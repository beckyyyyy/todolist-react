# React 專案

練習使用 React 進行 CRUD 功能實作。

## 開始使用

1. 請先確認已安裝 node.js 及 npm
2. 將專案下載到本地
3. 透過終端機進入該資料夾，請接著輸入：
   ```sh
   npm start
   ```
4. 若成功運行，將自動開啟瀏覽器連至網址如下：  
   http://localhost:3000

5. 功能頁對應網址：  
   [Login Page](http://localhost:3000/login)  
   [Sign up Page](http://localhost:3000/signup)  
   [Todo list Page](http://localhost:3000/todo)
